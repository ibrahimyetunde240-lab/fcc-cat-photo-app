# Cat Photo App

This project was completed as part of the freeCodeCamp Responsive Web Design curriculum.

## Overview

The Cat Photo App is a beginner HTML project that introduces the fundamentals of webpage structure and semantic HTML. It includes images, links, lists, and other essential HTML elements.

## Technologies Used

- HTML5

## Features

- Semantic HTML structure
- External links
- Images with descriptive alt text
- Ordered and unordered lists
- Figure and figcaption elements

## What I Learned

- HTML document structure
- Semantic HTML
- Hyperlinks
- Images and accessibility
- Lists and figures
- Organizing webpage content
